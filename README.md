# Задание для ZeBrains

Этот репозиторий содержит тестовое задание для ZeBrains.
Веб сервис сделан на Flask в связи с простотой использования.
Сервис имеет 3 эндпоинта, описанные ниже.

# 1. /emotion-detection
Использована модель cointegrated/rubert-tiny2-cedr-emotion-detection с huggingface. Модель представляет из себя 
рускоязычный вариант bert предобученный на данных для выявления эмоции в тексте.

Эндпоинт принимает аргумент text в качестве параметра и возвращает json с классами и их вероятностями в ответ.


# 2. /text-recognition
Выбран фреймворк easyocr за простоту установки и настройки и наличие русского и английского языков в списке доступных.

Эндпоинт принимает файл image в качестве параметра и возвращает весь распознанный текст с картинки одной строкой.

# 3 /similar-recognition
Использована модель setu4993/LaBSE в качестве модели векторного представления текста и реализация косинусного расстояния из pytorch. Модель выбрана за 
независимость от языка.
